## QUESTION:
A shop keeper would like to welcome their customers with their name.

Write a java program to get name from the user (String) and print it.

## AIM:
To write a java program to get name from the user (String) and print it.

## ALGORITHM :

1.	Start the program.
2. Create a Scanner object to read input from the user.
3. Read the customer’s name as a string.
4. Store the input in a variable name.
5. Display the message: "Hello, " followed by the customer’s name.
6. End the program.


## PROGRAM:

```
Developed by: Vikash S
RegisterNumber: 212222240115

import java.util.Scanner;

public class Main
{
    public static void main(String[] argv)
    {
        Scanner sc = new Scanner(System.in);
        String name = sc.nextLine();
        System.out.println("Hello, "+name);
    }
}
```
